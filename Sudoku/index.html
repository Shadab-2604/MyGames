How to Play Sudoku

  1. Fill in the empty cells with numbers.
  2. Each row must contain the numbers 1-9 (or 1-4 for 4x4, 1-16 for 16x16) without repeats.
  3. Each column must contain the numbers 1-9 (or 1-4 for 4x4, 1-16 for 16x16) without repeats.
  4. Each 3x3 subsection (or 2x2 for 4x4, 4x4 for 16x16) must contain the numbers 1-9 (or 1-4 for 4x4, 1-16 for 16x16) without repeats.
  5. The game is complete when all cells are filled correctly.
